5moraCVVv Cantonese Expansion Whole-Tone Auto-Splitting Dictionary for OpenUtau_ver1.0
by Aldof

Based on Leka's 5moraCVVv Cantonese Expansion Whole-Tone Recording Table, this enables auto-splitting for Cantonese voicebanks using this recording scheme in OpenUtau.
Download Leka's 5moraCVVv Cantonese Expansion Whole-Tone Recording Table: https://rekacurry10.wixsite.com/website/tool

This YAML file references this repository: https://github.com/stakira/OpenUtau/pull/776

How to use:

1. Copy the zhcvv.yaml file from the extracted folder into the directory of a Cantonese voicebank in OpenUtau that uses the 5moraCVVv Cantonese Expansion Whole-Tone recording scheme. For example, paste zhcvv.yaml into E:\OpenUtau\Singers\Weiming_Cantonese.

2. In OpenUtau, select the “ZH CVV (October-style Whole-Tone Expansion)” phonemizer.

3. Refresh the singer, then directly input Jyutping in the editor for auto-splitting.

Note: This version 1.0 only supports inputting Jyutping and using the October phonemizer for auto-splitting. The author Aldof is currently submitting a pull request to OpenUtau’s Cantonese phonemizer to add custom dictionary functionality. Version 2.0 may integrate automatic mapping of Chinese characters to Jyutping.


5moraCVVv広東語拡張整音自動分音辞書 for OpenUtau_ver1.0
by Aldof

Leka氏の5moraCVVv広東語拡張整音録音表に基づいて作成され、OpenUtauにおいてこの録音方式を使用する広東語音源に対し自動分音を実現します。
Leka氏の5moraCVVv広東語拡張整音録音表のダウンロード：https://rekacurry10.wixsite.com/website/tool

このyamlファイルは以下のリポジトリを参考にしています：https://github.com/stakira/OpenUtau/pull/776

使用方法：

1、解凍後のフォルダ内のzhcvv.yamlを、OpenUtauで5moraCVVv広東語拡張整音方式を使用する広東語音源のフォルダにコピーします。例：zhcvv.yamlをE:\OpenUtau\Singers\Weiming_Cantoneseに貼り付けます。

2、OpenUtauで「ZH CVV（十月式整音拡張）」フォネマイザーを選択します。

3、歌手を再読み込みし、編集画面で広東語拼音（Jyutping）を直接入力すると自動分音されます。

注意：この1.0バージョンでは、広東語拼音（Jyutping）の入力と十月式フォネマイザーの選択による自動分音のみ対応しています。作者Aldofは現在、OpenUtauの広東語フォネマイザーにカスタム辞書機能を追加するプルリクエストを提出中です。2.0バージョンでは、漢字から広東語拼音への自動マッピング機能が統合される可能性があります。
