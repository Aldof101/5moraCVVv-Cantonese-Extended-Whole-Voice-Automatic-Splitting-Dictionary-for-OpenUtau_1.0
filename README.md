5moraCVVv粤语扩张整音自动拆音词典 for openutau_ver1.0
by Aldof
基于Leka的5moraCVVv粤语扩张整音录音表制作，可以在openutau中对使用该录音方案的粤语音源实现自动拆音
Leka的5moraCVVv粤语扩张整音录音表下载：https://rekacurry10.wixsite.com/website/tool
此yaml文件参考了这个仓库：https://github.com/stakira/OpenUtau/pull/776
使用方法：
1、将解压后文件夹里的zhcvv.yaml复制粘贴到openutau里使用syo严式中文方案音源的文件夹下，例如将zhcvv.yaml粘贴到E:\OpenUtau\Singers\Weiming_Cantonese里
2、openutau里选择ZH CVV（十月式整音扩张）音素器
3、刷新歌手，在编辑界面直接输入粤拼即可自动拆音

注意：此1.0版本只能输入粤拼（Jyutping）和选择十月音素器进行自动拆音，作者Aldof正在对openutau的粤语音素器提交pull request以新增自定义词典功能，届时2.0版本可能整合自动输入汉字映射到粤拼的功能。


5moraCVVv Cantonese Extended Whole-Voice Automatic Splitting Dictionary for OpenUtau_ver1.0
by Aldof
Based on Leka's 5moraCVVv Cantonese Extended Whole-Voice Recording Table, this dictionary enables automatic phoneme splitting for Cantonese voicebanks using this recording scheme in OpenUtau.
Download Leka's 5moraCVVv Cantonese Extended Whole-Voice Recording Table: https://rekacurry10.wixsite.com/website/tool
This YAML file references this repository: https://github.com/stakira/OpenUtau/pull/776

Instructions:

1. Copy the zhcvv.yaml file from the extracted folder into the directory of an OpenUtau voicebank that uses the Syo Strict Chinese scheme. For example, paste zhcvv.yaml into E:\OpenUtau\Singers\Weiming_Cantonese.

2. In OpenUtau, select the "ZH CVV (Syo Extended Whole-Voice)" phonemizer.

3. Refresh the singer list, then directly input Jyutping in the editor for automatic splitting.

Note: This version 1.0 only supports Jyutping input and the Syo phonemizer for automatic splitting. The author, Aldof, is submitting a pull request to OpenUtau's Cantonese phonemizer to add custom dictionary functionality. Version 2.0 may integrate automatic mapping of Chinese characters to Jyutping.


5moraCVVv広東語拡張全音自動分割辞書 for OpenUtau_ver1.0
by Aldof
Lekaの5moraCVVv広東語拡張全音録音表に基づいて作成され、OpenUtauでこの録音方式を使用する広東語音源に対して自動分割を実現します。
Lekaの5moraCVVv広東語拡張全音録音表のダウンロード：https://rekacurry10.wixsite.com/website/tool
このYAMLファイルは、以下のリポジトリを参考にしています：https://github.com/stakira/OpenUtau/pull/776

使用方法：

1、解凍後のフォルダ内のzhcvv.yamlを、OpenUtauでSyo厳密中国語方式を使用する音源のフォルダにコピーします。例：zhcvv.yamlをE:\OpenUtau\Singers\Weiming_Cantoneseに貼り付けます。

2、OpenUtauで「ZH CVV（Syo拡張全音）」フォネマイザーを選択します。

3、歌手リストを更新し、編集画面で広東語拼音（Jyutping）を直接入力すると自動分割されます。

注意：このバージョン1.0は、広東語拼音（Jyutping）の入力とSyoフォネマイザーでの自動分割のみに対応しています。作者のAldofは、OpenUtauの広東語フォネマイザーにカスタム辞書機能を追加するプルリクエストを提出中です。バージョン2.0では、漢字から広東語拼音への自動マッピング機能を統合する可能性があります。
